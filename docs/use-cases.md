# Use Cases

StoryTime implements five primary use cases mapped to dedicated documents:

| Use Case | Summary | Spec |
| --- | --- | --- |
| UC-001 | Generate a bedtime story directly from Home with quick controls and immediate shelf visibility. | [UC-001 — Quick Generate](./use-cases/UC-001-quick-generate.md) |
| UC-002 | Continue a series with stable `seriesId`, recap continuity, and Story Bible alignment. | [UC-002 — Series Continuation](./use-cases/UC-002-series-continuation.md) |
| UC-003 | Gate full narration behind parent approval while allowing teaser playback first. | [UC-003 — Parent Approval Flow](./use-cases/UC-003-parent-approval.md) |
| UC-004 | Provide kid-safe browsing via curated Recent and Favorites shelves in Kid Shelf mode. | [UC-004 — Kid Shelf Recent and Favorites](./use-cases/UC-004-kid-shelf.md) |
| UC-005 | Enforce tier duration/concurrency/cooldown limits and support checkout/webhook upgrades. | [UC-005 — Tier and Cooldown Enforcement](./use-cases/UC-005-tier-cooldown.md) |
